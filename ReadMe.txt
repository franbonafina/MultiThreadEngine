To install iTvUpdateEngine service:

-Copy this folder to C:\iTV Update Engine. Run regedit and add two string registry keys under HKLM\Software\iTvUpdateEngine as shown on Regedit.png
-Run reg.bat as Administrator
-iTvUpdateEngine will show up in Services applet
-Click on iTvUpdateEngine Service properties and set it to run as your local Administrator account.
-Now you can run iTvUpdateEngine as a service.
