To install MultiThreadEngine service:

-Copy this folder to C:\multiThreadEngine. 
  Run regedit and add two string registry keys under HKLM\Software\multiThreadEngine as shown on Regedit.png
-Run reg.bat as Administrator
-multiThreadEngine will show up in Services applet
-Click on multiThreadEngine Service properties and set it to run as your local Administrator account.
-Now you can run it as a service.
